# WUT-Beamer-Theme
从[Trinkle23897](https://github.com/Trinkle23897/THU-Beamer-Theme)清华主题中拷贝过来，修改了主题颜色，按照最新武汉理工大学[校标](http://dzb.whut.edu.cn/cycx/xxbs/)的配色要求进行相关计算得出。

```LaTeX
red = 0 -> 0
green = 72 -> 0.282
blue = 155 -> 0.608
```

> 1. 基本上是清华模板的简单修改。
